----------------------------------------README-------------------------------------
Follow these steps to create local database for our website.
1. Download xampp from 'https://www.apachefriends.org/index.html'
2.Install xampp in default settings.
3.After installation, open xampp control panel and turn on/start 'Apache' and 'Mysql'.
4.Go to your browser and type "http://localhost/phpmyadmin/" to open the database creator.
5.Click on 'new' button to create new database. Name the database as 'login'.
6.Import the file 'login_sql' using the import button. Your table is ready now.
7.Now paste all the files in 'C:\xampp\htdocs' directory.
8.Go to your browser and search for 'localhost' and press enter.
9.Now you can view our website.
